### JAVA application with Spring Boot

To Run the app <br />
1st----> start Docker containers: cd docker/;   docker-compose up -d;   cd ..<br />
2nd----> cd target/ <br />
3rd----> "java -jar airport-0.0.1-SNAPSHOT.jar" <br />
